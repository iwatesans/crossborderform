時間が経つにつれ three.js の bloom がかかる様子に設定しているが, bloomは「強化」を表現することを意図している.  
当モデルにおいては, 「同じ」（あるいは類似した）「越境形式」の「重ね合わせ」, あるいは他の「越境形式」からの「作用」に起因し, 「強化」は生じるという設定を行っている.  

  
※　上記の「」の付いた語は, モデル特有の語義を持つため, この「」を付けた.
